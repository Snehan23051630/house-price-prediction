About the Model

How PriceWise estimates your property value.

📐 Algorithm

PriceWise uses Ordinary Least Squares Linear Regression (via Scikit-Learn). The model learns a linear relationship between two input features and the sale price of a home.

🔢 Features

  • sqft Interior square footage — the dominant driver of price.
  • bedrooms Number of bedrooms — a secondary signal.

📊 Training Data

The model is trained on 13 synthetic data points ranging from 600 to 4,000 sq ft. In a production system this would be replaced with real MLS data.
